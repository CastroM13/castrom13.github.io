Análise de Percepção Pública e Tópicos Emergentes¶

Estudo de Mineração de Dados sobre Figuras Presidenciáveis Brasileiras (2026) no X (ex-Twitter)¶

Este notebook apresenta uma análise abrangente da percepção pública sobre figuras presidenciáveis para 2026, utilizando dados do X (Twitter).

Candidatos analisados:

  • Luiz Inácio Lula da Silva (Lula)
  • Tarcísio Gomes de Freitas (Tarcísio)

Objetivos:

  1. Quantificar e qualificar a percepção pública
  2. Mapear tópicos emergentes
  3. Analisar tendências temporais de sentimento
  4. Identificar eventos que impactam a reputação

# Carregar dados processados
data_dir = Path('../data/processed')

# Dados principais com sentimento
df = pd.read_csv(data_dir / 'tweets_sentiment.csv', parse_dates=['created_at'])

# Dados de tópicos
df_topics_lula = pd.read_csv(data_dir / 'tweets_topics_lula.csv', parse_dates=['created_at'])
df_topics_tarcisio = pd.read_csv(data_dir / 'tweets_topics_tarcisio.csv', parse_dates=['created_at'])

topic_sentiment_lula = pd.read_csv(data_dir / 'topic_sentiment_lula.csv')
topic_sentiment_tarcisio = pd.read_csv(data_dir / 'topic_sentiment_tarcisio.csv')

# Dados temporais
df_comparison = pd.read_csv(data_dir / 'temporal_comparison.csv', parse_dates=['created_at'])
df_hourly = pd.read_csv(data_dir / 'hourly_patterns.csv')
df_dow = pd.read_csv(data_dir / 'day_of_week_patterns.csv')

print(f'  Período: {df["created_at"].min().date()} até {df["created_at"].max().date()}')
